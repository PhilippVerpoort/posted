## `is_float`

is_float

### Description

Checks if a given string can be converted to a floating-point number in Python.


### Usage

```r
is_float(string)
```

### Arguments

Argument      |Description
------------- |----------------
`string` | Character. String to check.

### Return Value

Logical. `TRUE` if conversion was successful, `FALSE` if not.


### Examples

```r
## Example usage:
is_numeric("3.14")
```
